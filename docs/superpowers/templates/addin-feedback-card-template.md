# Шаблон карточки обратной связи по надстройке

1 файл = 1 замечание.

Если замечаний несколько, нужно сделать копию шаблона на каждое замечание отдельно.

Что важно:
- писать короткий и понятный заголовок проблемы или идеи;
- обязательно указывать, где именно это найдено: вкладка, кнопка, форма, сценарий, файл;
- для багов обязательно описывать шаги воспроизведения и фактический результат;
- если есть скриншот, пример файла или короткое видео, их нужно приложить.
