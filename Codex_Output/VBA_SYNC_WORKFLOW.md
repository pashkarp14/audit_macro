# Синхронизация VBA между двумя ПК (без OneDrive)

## Что это даёт
- VBA-код хранится в обычных файлах (`.bas`, `.cls`, `.frm`) + код модулей листов и `ThisWorkbook`.
- Изменения удобно смотреть построчно.
- Код можно безопасно переносить между двумя компьютерами.

## Файлы и папки
- `modVbaSync.bas` — модуль, который нужно импортировать в проект VBA книги.
- Папка синхронизации (создаётся автоматически): `<ПапкаКниги>\_vba_sync`
- Внутри:
- `modules\`
- `classes\`
- `forms\`
- `documents\`
- `manifest.tsv`

## Обязательная настройка Excel
- Включи доступ к объектной модели VBA:
- `Файл -> Параметры -> Центр управления безопасностью -> Параметры центра -> Параметры макросов -> Доверять доступ к объектной модели проектов VBA`

## Основные макросы
- `ExportAllModules` — экспортирует все компоненты книги в `_vba_sync`.
- `ImportAllModules` — импортирует компоненты обратно из `_vba_sync`.
- `ExportAllModules_UI` — экспорт с выбором папки через диалог.
- `ImportAllModules_UI` — импорт с выбором папки через диалог и подтверждением.

## Как повесить на кнопки
1. На листе Excel: `Разработчик -> Вставить -> Кнопка (элемент формы)`.
2. Назначь макросы:
- для кнопки экспорта: `ExportAllModules_UI`
- для кнопки импорта: `ImportAllModules_UI`
3. Переименуй подписи кнопок, например: `Экспорт VBA`, `Импорт VBA`.

## Рекомендуемый процесс (2 компьютера)
1. На рабочем ПК:
- открыть книгу
- запустить `ExportAllModules`
- сделать `commit/push` папки `_vba_sync` в git (или архивировать и передать вручную)

2. На домашнем ПК (где работаешь с ассистентом):
- получить свежую версию (`git pull` или распаковать архив)
- править файлы в `_vba_sync`
- сделать `commit/push`

3. Снова на рабочем ПК:
- получить изменения (`git pull` или копирование обновлённой папки)
- открыть книгу
- запустить `ImportAllModules`
- протестировать макросы в книге

## Важные моменты
- Модуль `modVbaSync` защищён от удаления во время импорта.
- Модули документов (`ThisWorkbook` и код листов) обновляются заменой текста кода.
- Если VBA-проект запаролен/заблокирован, импорт и экспорт не сработают.
