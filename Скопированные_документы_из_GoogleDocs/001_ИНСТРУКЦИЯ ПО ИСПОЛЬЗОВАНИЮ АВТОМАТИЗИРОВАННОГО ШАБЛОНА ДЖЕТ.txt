﻿Чтобы шаблон работал, не нужно добавлять нигде строки, столбцы, удалять\менять названия листов и тд., все что нужно делается автоматически. На листе “Панель управления” находится кнопка, которая собирает весь документ. При изменении любой настройки выборки все пересобирается автоматически. Так же есть возможность раскрыть кнопки, +собирающие какие-то части РД (для продвинутых пользователей). Если не идет какая-то сверка, в 95% случаев это значит что-то не так с выгрузками. Не забывайте вставлять правильный уровень существенности. Заполнить выгрузками нужно только те листы, которые указаны ниже. Остальные заполняются автоматически уже  обработанными  данными. В выгрузках не должно быть фильтров,с ними часть данных может потеряться (на это есть защита). При первом использовании советую открыть уже заполненный шаблон, чтобы посмотреть куда какие выгрузки вставлять.
Для сборки РД Джет нужно скопировать на соответствующие листы шаблона следующие выгрузки:
1. Журнал проводок (если журнал состоит менее чем из 1 048 576 строк, вставляем его на один лист “ЖП”, если больше, предусмотрена возможность разделения журнала на сколько угодно листов. Листы называть “ЖП_Ч1”, “ЖП_Ч2” и так далее)
2. ОСВ за необходимый период
3. Журнал операций за необходимый период (для установления ответственных по ручным проводкам)
4. Заполнить список ВГО
Есть возможность менять любые настройки выборок, менять ключевые слова для теста слов, менять дни для теста праздников. На листе “2. Проводки к исключению” можно менять меппинг любой проводки, он автоматически изменится во всех журналах проводок. Более подробные инструкции написаны на листе “Панель управления”, так же рекомендую прочитать.
Далее переходим  на лист “Панель управления” и нажимаем большую кнопку “Собрать рабочий документ”. После этого можно пройтись по  листам и посмотреть на получившиеся выборки. Если не устраивает какая-нибудь выборка, стираем сид из ячейки “Random seed”, выборка переделывается, все необходимые листы перезаполнятся автоматически.
